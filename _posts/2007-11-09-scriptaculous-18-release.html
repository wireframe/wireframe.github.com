With the recent release of the scriptaculous 1.8 javascript library, I've released an update to the wicketstuff-scriptaculous project that incorporates these latest changes (including prototype 1.6!).



No Java API changes are in this release, so it should be a transparent upgrade for users. See the scriptaculous release notes for a complete list of changes.
