I always knew that working on open source projects was an investment of time, but now I can see how much it would cost! Koders.com has estimated development costs for shard, based on some basic formulas like: LINES_OF_CODE x COST_PER_1000_LINES. To be honest, it's a little lower than what I expected. They don't have all of the source code available yet (shard-minotaur is not there), so those numbers might increase once they have the complete project.


I'll have to be careful who I show this to. I don't think my wife was too impressed when she thought of how much I've invested in this project that could have been spent doing more freelance consulting.
